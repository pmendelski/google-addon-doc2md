Doc2Md

Converts content of Google doument to Markdown format. Project is open sourced and available on GitHub.

How to use the AddOn?

Simply click the "Convert" button in the Doc2Md Sidebar. Generated Markdown is available in the textfield. Before using the output make sure to resolve all warnings.

Why images are not converted to Markdown?

It is not possible to retrieve image url from google doc api. Therefore images must be fixed manually.
